## Kurulum
### Gerekli Paketler

```sh

meteor npm install
 ```

### settings.json
```javascript
{
  "private": {
    "apiKey": "YOUR_API_KEY_HERE"
  }
}
 ```
### Uygulamayı Yürütme
```javascript

meteor --settings settings.json
```
